Projects

Some of the Projects I have been part of:

Credit Default Risk (Statistics and Machine Learning Project)

Predicted Credit Default risk from given testing and training data in a team of 3. Python was used for LogisticRegression, Gradient Boosted Trees, Lasso Regression, and Final Model Evaluation and Implementation. Implemented Univariate Analysis in Tableau.

Forecasting Stock prices using VAR Stock Prices (Financial Time Series Course Project)

Forecasted the stock prices of the Healthcare and Energy sector and the Dow Jones Index using Vector Autoregression model and Vector Error Correction Model in a team of 3. Tested the forecasted values and got very low RMSE values of these stock returns suggesting accurate results.

Prediction of S&P 500 Index (Object-Oriented Programming Course Project)

Predicted The S&P 500 index using Neural Networks in C++ programming language in a team of 3 using a one input layer-one hidden layer Multilayer Neural Network Model. Most of the predictions were less than 5% error in price.

Hedge Fund Project (Hedge Funds Course Project)

Managed an imaginary asset management firm that used a Long/Short Equity strategy over a period to give an alpha of a whopping 1.12% per year.
